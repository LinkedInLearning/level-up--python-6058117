# Python 代码挑战 #5: 玩等待游戏

你的目标是实现函数 `waiting_game()`，该函数会先显示一条消息，提示玩家等待2到4秒之间的随机秒数。当玩家按下 回车 键时，计时器开始。玩家的目标是等待指定的秒数后，再次按下 回车 键。然后，函数会显示一条消息，显示实际经过的时间，告诉玩家表现得如何：快了、慢了还是完美，并且显示实际经过时间与目标时间之间的差异。

## 示例测试输出
```console
>>> waiting_game()

你的目标时间是 2 秒
 --- 按 Enter 键开始 ---

...等待 2 秒后再次按 Enter 键...

经过时间：1.680 秒
（快了 0.320 秒）
```
